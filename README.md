# Library in nodejs and marko

## Quick start
Generator sqlite
~~~script
yarn database.js
~~~~

Start aplication
~~~script
yarn start
~~~~

#### GET
```
    /
    /livros
    /livros/form
    /livros/form/:id
    /usurios
    /usurios/form
    /usurios/form/:id
```
#### POST
```
    /livros
    /usurios
```
#### PUT
```
    /livros
    /usurios
```
#### DELETE
```
    /livros/:id
    /usurios/:id
```
